# Installation

### Pre-Requisites on OCI Instance
If you are on OCI, provision a compute instance with Oracle Linux 9. Please refer to online documentation on how to create an OCI Compute: https://docs.oracle.com/en-us/iaas/Content/Compute/Tasks/launchinginstance.htm
### Pre-Requisites on VM/Bare Metal
If you are not on OCI, provision a server in VM/Bare Metal with Oracle Linux 9 or RedHat Enterprise Linux 9.
